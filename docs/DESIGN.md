# Дизайн-система «уДачный сад» — v5.1 «Садовая книжка»

Статус: на утверждении. После апрува — применить к коду (§10).
Изменение v5.0 → v5.1: объекты схемы переведены в стиль генплана (§3.2, §3.4).

---

## 0. Направление

**Советский учёт и агитплакат.** Продукт — про накопление знаний об участке,
поэтому визуальный язык — язык великого документооборота: **садовая книжка**,
учётные карточки, журналы учёта, круглые печати, вымпелы и плакатные лучи солнца.
Пожелтевшая бумага, кумач, золото колосьев, чернила и машинопись.

Сигнатуры продукта:
- **Круглая печать** — подтверждения действий: оттиск «ПОСАЖЕНО ★» по кругу
  «уДачный сад · журнал учёта садовода», красный, с поворотом −8°;
- **Схема участка = генплан по ГОСТ** — условные обозначения как на советских
  чертежах: зубчатые кроны, штриховки, волны воды, экспликация с номерами,
  угловой штамп листа, стрелка севера, масштабная линейка (§3);
- **Учётная карточка растения** — форма документа: «УЧЁТНАЯ КАРТОЧКА № 042»,
  поля с подчёркиваниями, значения машинописью, уголок «место для фотокарточки»;
- **Вымпел «УДАРНИК УЧЁТА»** — треугольный флажок за стрик записей;
- **Плакатное небо** — лучи агитплаката по времени суток;
- **Лозунги** — пустые состояния и заголовки: «НИ ОДНОЙ ГРЯДКИ БЕЗ ЗАПИСИ!».

Символика — ностальгическая (звезда, колос, печать, тетрадь, чертёж),
без государственной и идеологической (герб, серп и молот не используются).

Стиль полярный и очень характерный — проверить на первых пользователях (этап 3 плана).
Контрасты и крупные цели касания — жёсткие требования, как во всех версиях.

## 1. Принципы

1. **Всё — документ.** Двойные рамки, номера, печати, штриховки по ГОСТ.
2. **Кумач — один и главный.** Красный ведёт действие; золото и зелень — поддержка.
3. **Читаемость на солнце.** Текст ≥ 4.5:1, базовый 17px.
4. **Крупные цели касания.** ≥ 48×48 px, кнопки 56, зазоры ≥ 8px.
5. **Наклон — только печатям и вымпелам** (±3–8°). Текст, кнопки и чертёж — по линейке.
6. **Цвет не единственный носитель смысла** (зоны и обозначения: заливка + фактура + подпись).

## 3. Схема участка — генплан

### 3.1 Лист

- Бумага: `paper`; клетка — линии `grid-blue` 1px с шагом 1 м, каждая 5-я
  `#B9C7D6` 1.2px (тетрадь-миллиметровка).
- Рамка листа: двойная — `ink` 3px + внутренняя 1px с отступом ~5px.
- За рамкой: `#EFE5C9`.

### 3.2 Объекты — условные обозначения (по мотивам ГОСТ)

Все объекты: контур `ink` 2.5px + заливка + **чертёжная фактура** (не блики):

| Тип | Заливка | Фактура-обозначение |
|---|---|---|
| Газон | `#E9F0CF` | «галочки» травы `#7FA457` (паттерн, шаг ~26px); без контура |
| Грядка | `#C08A57` | параллельные бороздки `#8F6741` («пашня») |
| Клумба | `#E8A0AC` | точечная заливка `#C4747F` («цветник») |
| Дорожка | `#E6DCC0` | точки-гравий `#B8AA82` |
| Постройка | `#C9C2AC` | диагональная штриховка `ink` 55% («капитальное строение») |
| Теплица | `#E2EEE9` | сетка остекления `#6E8F86` (обе диагонали) |
| Вода | `#A9CFE2` | горизонтальные волнистые линии `#3E6E8E` |
| Дерево | `#6FAE52` | **зубчатая крона** (фестончатый контур) + точка ствола `ink` в центре |
| Кустарник | `#8FC468` | зубчатый контур малого радиуса |
| Калитка/ворота | `#C9C2AC` | без штриховки; иконка ворот поверх точки; контур `ink` 2.5px |

- Зубчатая крона: контур из наружных дуг; число зубцов ≈ радиус_px / 3.2 (мин. 8);
  радиус дуги ≈ хорда / 1.7. В Konva — кастомный `sceneFunc`.
- **Выбранный объект:** кольцо `red` 4px — обведено красным карандашом ревизора.
- Подписи на плане не пишутся — вместо них **номера** (§3.4).

> Сверить перечень с enum `type` в `schemaObjects` (ARCHITECTURE.md).

### 3.3 Зоны условий (проверенные, без изменений)

| Зона | Заливка | Граница (пунктир 2px) | Приём |
|---|---|---|---|
| Солнце | `rgba(255,214,74,.14)` | `rgba(214,166,26,.85)` | — |
| Полутень | `rgba(90,105,120,.24)` | `rgba(90,105,120,.85)` | — |
| Тень | `rgba(42,56,70,.42)` | `rgba(42,56,70,.9)` | Штриховка |
| Сухо | `rgba(217,164,74,.22)` | `rgba(191,138,48,.85)` | Точечный пунктир |
| Умеренно | `rgba(95,168,211,.18)` | `rgba(95,168,211,.85)` | — |
| Влажно | `rgba(46,111,163,.34)` | `rgba(46,111,163,.9)` | Штриховка |

Одновременно виден один слой.

### 3.4 Атрибуты генплана

- **Номера объектов:** кружок ⌀20 `paper` с рамкой `ink` 1.5px, номер PT Mono 11/700.
  Одинаковые объекты (три яблони одного сорта) — один номер на группу.
- **Экспликация:** легенда-таблица «№ — наименование — примечание» (PT Mono №,
  системный текст); в приложении — выдвижная панель под схемой; касание строки
  подсвечивает объект на плане (кольцо `red`).
- **Угловой штамп листа** (правый нижний угол): рамка `ink` 1.5px, строки:
  «УДАЧНЫЙ САД» (Oswald 10) / «ГЕНПЛАН УЧАСТКА» (Oswald 12/700) /
  «М 1:100 · ЛИСТ 1 · {год}» (PT Mono 9.5). Масштаб вычислять из canvasConfig.
- **Стрелка севера:** тонкая стрелка `ink` + буква «С» (ориентация — из настроек
  участка; по умолчанию вверх). В MVP — статична.
- **Масштабная линейка:** чередующиеся чёрно-белые сегменты по 1 м с подписями
  PT Mono (левый нижний угол).
- Штамп, север и линейка скрываются при зуме ближе ~1:50, чтобы не мешать работе.

## 5. Форма, отступы, касания

- Шаг 4px; типовые 8 / 12 / 16 / 20; боковые поля 16.
- **Двойная рамка** (документ): внешняя 2px `ink` + внутренняя 1px с отступом 4–5px.
  Скругления малые: кнопки 8, карточки 10, вкладки 6, sheets 14 (верх).
- Высота: кнопки 56, поля 52, строки реестра 56.
- Тень `blank`; нажатие — сдвиг на 2px вниз, тень короче.
- Наклон: печати −8°, вымпел +3°.

## 6. Компоненты

**Кнопки** (радиус 8, высота 56, Oswald 16/600 КАПС, двойная рамка, тень `blank`):
- Primary: `red`, белый текст: «ПОСАДИТЬ». Главное действие — во всю ширину.
- Secondary: `paper`, текст `ink`: «ОТМЕНА».
- Danger: `paper`, двойная рамка `red`, текст `red`: «СПИСАТЬ»; всегда с подтверждением.
- Plain: текст `blue-ink` с подчёркиванием.
- Disabled: `#EAE2CC`, текст `ink-muted`, без тени.

**Учётная карточка растения:** бланк `surface` с двойной рамкой и тенью; шапка
«УЧЁТНАЯ КАРТОЧКА № 042»; поля: ярлык 13/600 КАПС `ink-muted`, значение PT Mono 15
`blue-ink` на пунктирном подчёркивании; рамка 3×4 «место для фотокарточки».

**Печать (подтверждения):** круглый SVG-оттиск ⌀110: два кольца `red`, текст по кругу
«УДАЧНЫЙ САД · ЖУРНАЛ УЧЁТА САДОВОДА», в центре действие КАПСОМ + ★; поворот −8°,
появление «ударом» (scale 1.2 → 1). `prefers-reduced-motion` — без анимации.

**Вымпел (стрик):** флажок `red` с кантом `gold` и ★: «УДАРНИК УЧЁТА · 5 ДНЕЙ»,
Oswald 12 КАПС белым, наклон +3°.

**Экспликация (легенда схемы):** выдвижная панель-бланк; строки «№ · наименование ·
примечание»; касание — подсветка объекта.

**Реестр (списки):** строки-бланки 56px с рамкой 1.5px, № PT Mono; заголовок секции
«РАЗДЕЛ I. ДЕРЕВЬЯ» Oswald 14.

**Поля:** `surface`, рамка 2px `ink`, радиус 8; ярлык 13/600 КАПС; ввод PT Mono;
фокус — рамка `red` + тень; ошибка — подпись `red`.

**Sheets:** снизу, радиус 14, двойная рамка сверху, ручка 40×5 `ink`.

**Иконки:** lucide-react 24/2px `ink`, всегда с подписью (кроме зум/отмена, aria-label).

## 7. Тексты интерфейса

Тон: тёплая канцелярия с юмором — лозунги КАПСОМ, остальное человеческим языком на «вы».

| Плохо | Хорошо |
|---|---|
| Add plant | Посадить растение |
| History | Что росло здесь (архивная справка) |
| Success | Печать «ПОСАЖЕНО ★» |
| Streak | УДАРНИК УЧЁТА · 5 ДНЕЙ |
| Delete object? | Списать грядку со схемы? Записи в журнале сохранятся |
| Error 500 | Не получилось сохранить. Проверьте интернет и попробуйте ещё раз |
| Пустой экран | НИ ОДНОЙ ГРЯДКИ БЕЗ ЗАПИСИ! Добавьте первое — дом или грядку |

Лозунги — только в заголовках и пустых состояниях.

## 8. Доступность

- Контраст: `ink` на `paper`/`surface` ≥ 10:1; белый на `red` ≥ 5:1;
  `gold` — не для текста; `blue-ink` на `paper` ≥ 6:1.
- Цели касания ≥ 48px; `focus-visible` — рамка `red` 3px.
- Объекты различимы фактурой, не только цветом (штриховки, волны, зубцы) +
  номера с экспликацией; зоны — пунктир + штриховка + подпись.
- `prefers-reduced-motion`: печать без «удара», кнопки без сдвига.
- КАПС — только короткие строки (до 4–5 слов).

## 9. Что убрано из v5.0

## 10. Внедрение (задача для разработки)

1. `npm i @fontsource/oswald @fontsource/pt-mono`, подключить в корне.
2. Токены §2 в `tailwind.config` (цвета, boxShadow.blank, fontFamily poster/mono).
3. `src/theme/canvasColors.ts` + `src/theme/canvasPatterns.ts` — §3 для Konva:
   клетка, заливки, фактуры (штриховки, волны, галочки травы, точки),
   `sceneFunc` зубчатой кроны (параметры из §3.2), выделение red.
4. `src/theme/sky.ts` — плакатные лучи по времени суток (§2).
5. Компоненты §6: печать (SVG), вымпел, учётная карточка, экспликация, реестр,
   кнопки с двойной рамкой.
6. Атрибуты генплана §3.4: номера объектов, угловой штамп (масштаб из canvasConfig),
   стрелка севера, масштабная линейка, скрытие при близком зуме.
7. Экраны этапа 1 → §4–6; шапка участка — лучи + плашка-бланк.
8. Тексты §7, доступность §8. Ничего сверх перечисленного не менять.
