# PLAN11 — Переориентация на ДЕКОРАТИВНЫЙ САД

> **ВАЖНО:** уДачный сад — это приложение для учёта **декоративного сада/цветника/ландшафтного дизайна**.
> НЕ огорода. Никаких грядок, теплиц, урожая, плодовых культур, овощей.
> Фокус: клумбы, миксбордеры, композиции, живые изгороди, декоративные деревья и кустарники, многолетники.

## Контекст для Claude

- **Стек:** React + TypeScript + Vite PWA, PocketBase backend, Tailwind CSS, Canvas
- **Путь к коду:** `/root/projects/udachny-sad/app/`
- **PocketBase:** работает на `localhost:8090`, URL фронта: `https://pb.kdnfx.space`
- **Вход по email** уже работает (PocketBase emailPassword auth)
- **Гес делает инфру:** PB admin API (изменение коллекций), nginx, systemd, билд и деплой
- **Claude делает код:** все `.ts/.tsx` файлы

## Текущее состояние

Приложение полностью работает на PocketBase (Convex убран), все экраны мигрированы.
PocketBase schema включает `schemaObjects` с типами: `building | greenhouse | lawn | path | bed | flowerbed | tree | shrub | water | gate | other`
